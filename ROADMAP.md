# Roadmap

Reflexor is currently in an early scaffolding phase. The focus so far is on:

- Modern Python packaging (`pyproject.toml`, `src/` layout)
- Linting/formatting/type checking/test tooling
- CI (GitHub Actions) and basic repo hygiene

## Next (planned)

- Define core abstractions: reflex, planner, executor, tools, and policy.
- Implement deny-by-default tool access and a dry-run mode for side effects.
- Add examples and documentation for intended workflows.
- Expand test coverage and raise the coverage threshold over time.

## Tracking

Roadmap items are expected to move into GitHub issues/PRs as they become actionable.
